* NixOS Template

** Usage

Update inputs

#+begin_src shell
nix flake update
#+end_src

Rebuild and switch (this is what you need 99% of the time)

#+begin_src shell
nixos-rebuild switch --flake . --use-remote-sudo
#+end_src

Build

#+begin_src shell
nixos-rebuild build --flake .
#+end_src
